เรื่องที่คุณ(อาจ)ยังไม่รู้เกี่ยวกับ “IE”

1
Internet Explorer 1 เปิดตัวเมื่อวันที่ 16 สิงหาคม ค.ศ.1995 โดยเป็นรุ่นพัฒนาจากการซื้อบราวเซอร์ Mosaic ของบริษัท Spyglass มาเป็น Internet Explorer ของ Microsoft

2
ในช่วงปี ค.ศ.2002-2003 Internet Explorer มีส่วนแบ่งการตลาดถึง 95% หลังจากการมาของ Chrome และ Firefox ก็เหลือเพียง 17% เท่านั้น


SOURCE

3
Microsoft เคยพัฒนา Internet Explorer ให้ระบบปฏิบัติการ Mac ในปี ค.ศ.1996 แต่อีก 7 ปีให้หลัง Microsoft ก็เลิกพัฒนาไป


SOURCE

4
ในปัจจุบัน Internet Explorer ได้เลิกพัฒนาบนทุกระบบปฏิบัติการแล้วโดยมีรุ่น 11 เป็นรุ่นสุดท้าย เปิดตัวเมื่อวันที่ 11 มีนาคม ค.ศ.2015


SOURCE

5
Internet Explorer เริ่มพัฒนาขึ้นโดยนักพัฒนาเพียง 6 คนเท่านั้น


SOURCE

6
ในปี ค.ศ.1999 ถือเป็นยุคทองของ Internet Explorer เพราะ Microsoft ทุ่มงบพัฒนากว่า 3,000 ล้านบาทต่อปี พร้อมด้วยนักพัฒนาอีกกว่า 1,000 คน


SOURCE

7
หลังจากการที่ Microsoft ตัดสินใจรวม Internet Explorer เข้ากับ Windows 95 Microsoft ได้ถูกอัยการจาก 20 รัฐทั่วสหรัฐฯฟ้อง ถึงแม้ศาลจะไม่ได้ตัดสินให้ Microsoft แพ้แต่ก็ส่งผลเสียต่อภาพลักษณ์ของ Internet Explorer มาถึงทุกวันนี้


SOURCE

8
